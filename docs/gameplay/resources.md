# Resources

> Dieses Dokument beschreibt alle Ressourcentypen und ihre Rolle in der Wirtschaft von Project Genesis.

Version: 1.0

---

# Grundprinzipien

Ressourcen bilden die Grundlage der gesamten Wirtschaft.

Nahezu jedes Gebäude, jede Maschine und jedes Produkt entsteht aus Ressourcen.

Jede Ressource besitzt mindestens eine Verwendung.

Es existieren keine reinen Dekorationsobjekte.

---

# Ressourcenklassen

# Energie

Strom ist eine eigene Ressource.

Er kann:

- produziert
- gespeichert
- verbraucht

werden.

Strom wird nicht dauerhaft gelagert.

Später können Batteriespeicher gebaut werden.

# Erweiterbarkeit

Neue Ressourcen müssen lediglich definieren:

- Kategorie
- Produktionsgebäude
- Eingänge
- Ausgänge

Dadurch können jederzeit neue Industrien ergänzt werden.

---

# Designregeln

✔ Jede Ressource besitzt mindestens eine Verwendung.

✔ Jede Produktionskette ist vollständig.

✔ Keine Sackgassen.

✔ Keine Ressource existiert ausschließlich zum Verkauf.

✔ Komplexität entsteht durch Kombination vieler einfacher Produktionsschritte.
