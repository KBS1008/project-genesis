# Color Palette

**Project:** Project Genesis  
**Version:** 1.0  
**Status:** Planning  
**Last Updated:** 2026-07-12

---

# Purpose

This document defines the official color palette of Project Genesis.

The goal is to establish a consistent visual identity across the entire game while ensuring excellent readability, accessibility and long-term maintainability.

Every graphical asset should follow this document.

---

# Design Philosophy

Project Genesis is an economic simulation.

The visual style should communicate:

- professionalism
- clarity
- stability
- trust
- realism

The interface should never overwhelm the player with unnecessary visual effects.

The simulation itself is the focus.

---

# Design Goals

The palette should be:

- modern
- neutral
- timeless
- highly readable
- accessible
- color-blind friendly whenever practical

# Accessibility

The palette should:

- maintain sufficient contrast
- avoid relying solely on color
- use icons together with colors where possible
- support common forms of color blindness

Text should meet WCAG AA contrast recommendations.

---

# Future Themes

The architecture should support additional themes.

Examples:

- Light Theme (default)
- Dark Theme
- High Contrast Theme
- Color Blind Theme

Theme switching should not require changes to business logic.
