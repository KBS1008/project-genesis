Neue Architekturidee (DD-019 – Technology Tree)

Beim Schreiben ist mir eine Idee gekommen, die das Spiel langfristig deutlich interessanter machen könnte.

Ich würde keinen klassischen Forschungsbaum verwenden, sondern einen Technologie-Graphen.

Statt:

A → B → C → D

könnte ein Spieler verschiedene Wege einschlagen:

                Energie
               /       \
Produktion ── Logistik ── Automatisierung
               \       /
             Management

Dadurch entstehen unterschiedliche Unternehmensprofile:

Ein Unternehmen spezialisiert sich auf Massenproduktion.
Ein anderes auf Energieeffizienz.
Ein drittes auf Logistik und Handel.

Das erhöht den Wiederspielwert erheblich und verhindert, dass am Ende alle Unternehmen identisch sind. Diese Idee würde ich als Version 2 einplanen, damit Version 1 überschaubar bleibt.
